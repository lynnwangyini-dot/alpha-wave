# DESIGN.md

## 项目定位
Alpha 脑波音乐生成器。用户上传喜欢的歌曲，系统提取 BPM / 调性 / 情绪 / 能量曲线，**程序化合成** 一段完全原创的 ambient 音乐并嵌入 8–12Hz Alpha 双耳节拍。
强调"声学科学 × 极简科技"——像一台放在桌面上、安静呼吸的脑波设备。

## 气质与意象
**意象锚点**：深空舱内一盏悬浮的霓虹环，暗紫渐变里偶尔亮起一缕青色脉冲。
不是 SaaS 仪表盘的"蓝紫渐变 + 圆角卡片"那种通用 AI 审美。也不是传统音频工作站的密集推子界面。
**关键词**：极简、玻璃、深空、单色、脉冲、微光、参数外露、流体。

## 视觉策略
- **摄影/图像方向**：无图片素材。全部由 CSS / SVG / Canvas 实时生成（粒子、波形、雷达）。
- **图形语言**：等宽数字 + 几何线条 + 微弱辉光（glow） + 玻璃面板（backdrop-blur）。避免一切拟物。

## 配色方案
| 角色 | 色值 | 意象 |
| --- | --- | --- |
| Background | `#070712` | 真空深空，略带冷蓝 |
| Surface | `#0E0E1A` | 玻璃面板底 |
| Surface Elevated | `#14142B` | 上浮层 |
| Border | `rgba(255,255,255,0.06)` | 玻璃切边 |
| Primary (Violet) | `#8B5CF6` | 神经脉冲 |
| Accent (Cyan) | `#22D3EE` | 脑波谐振 |
| Accent (Rose) | `#F472B6` | 高能段预警 |
| Text Primary | `#E8E8F0` | 接近白但不到纯白 |
| Text Muted | `#7A7A8C` | 哑光灰紫 |

## 字体
- 全局等宽：`JetBrains Mono`（标题与数字）/ `Inter`（正文）
- 中文：`Noto Sans SC`
- 排版节奏：大量留白，标题字距 +0.04em，参数数字字距 -0.02em

## 圆角 / 阴影
- 圆角：主面板 24px，子卡片 16px，按钮 12px
- 阴影：极弱，仅在主面板下方 `0 24px 60px -30px rgba(139,92,246,0.45)`
- 发光：关键元素使用 `box-shadow: 0 0 40px rgba(139,92,246,0.35)` 制造脉冲

## 动效
- 缓动：`cubic-bezier(0.22, 1, 0.36, 1)`（sine out 风格）
- 关键帧：呼吸（3s 循环）、扫描（1.5s 线性）、脉冲（2.4s ease-in-out）
- 进度条粒子拖尾；按钮悬停时背景由透明渐变到 `rgba(139,92,246,0.08)`

## 页面结构
单页应用，垂直三段：
1. **Hero / 上传区**：品牌、标语、拖拽上传卡片、示例 MP3 试用
2. **分析结果面板**：BPM、调性、情绪、能量曲线（4 个等宽数字卡 + 折线图）
3. **生成 & 播放器**：Alpha 频率滑块（8–12Hz）、生成按钮、可视化播放器、下载按钮
底部固定状态栏：渲染状态 / 当前阶段

## 组件规范
- 所有数字指标使用大号等宽 + 微弱辉光
- 卡片背景 `bg-[#0E0E1A]/70 backdrop-blur-xl border border-white/[0.06]`
- 按钮：主按钮发光边框 + 内填透明；次按钮纯边框
- Slider：轨道细 2px，拇指带霓虹环

## 交互
- 上传后立即本地分析，无需等待后端
- 生成时显示 4 阶段进度：解码 → 合成 → 烘焙 → 完成
- 播放中画布实时显示双耳节拍相位 + 频谱
- 头部"试听中"标签呼吸闪动

## 响应式
- ≥1024：横向三段
- 640–1024：纵向堆叠，分析卡 2×2
- <640：单列，字号 14px 主，触控按钮 ≥44px

## 可访问性
- 所有色彩对比度 ≥ 4.5:1（正文）/ 3:1（大号数字）
- 关键操作提供键盘焦点环（青色 outline）
- 不依赖颜色传递状态（配合文字/图标）

## 设计与实现备注
- 音频分析 **必须纯客户端**（Web Audio API + Canvas 离线分析）以保护隐私与即时反馈
- 音乐生成 **必须基于程序化合成**（Oscillator + BiquadFilter + 噪声），**不引用原曲样本**，天然规避版权
- 双耳节拍使用 `ChannelMergerNode` 分离左右声道
- 下载使用 `OfflineAudioContext` 烘焙成 WAV（无后端编码依赖）

## 禁忌
- ❌ 蓝紫渐变 + 圆角白卡的通用 AI 风格
- ❌ 拟物推子、EQ 旋钮、磁带/黑胶图形
- ❌ Emoji 图标
- ❌ 任何拟人化卡通或写实摄影
- ❌ "Remix" 原曲或上传音频的任何片段
